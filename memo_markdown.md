# Markdown

#### Italique :
\*hello\* 

*hello*

---
#### Gras :
\*\*hello\*\* 

**hello**

---
#### Titres :
\# Mon titre niveau 1

\#\# Mon titre niveau 2

\#\#\# Mon titre niveau 3
...

Exemple :
# Mon titre niveau 1
## Mon titre niveau 2
### Mon titre niveau 3

--------

#### Listes à puces :
\* Puce 1

\* Puce 2

(TAB)\* Puce 2.1

(TAB)(TAB)\* Puce 2.1.1

Exemple :

* Puce 1
* Puce 2
   * Puce 2.1
      * Puce 2.1.1

-------

#### Listes numérotées :
1\. Num 1

2\. Num 2

2\.1 Num 2.1


Exemple :

1. Num 1

2. Num 2

   2.1 Num 2.1

------
#### Citations :
\> Une citation

> Une citation

---
#### Bloc de code :
(TAB)int main(){

(TAB)(TAB)printf("hello")

(TAB)}

Exemple :

    int main(){
        printf("hello")
    }
----
#### Code inséré :
Voici la fonction \`printf()\` !
